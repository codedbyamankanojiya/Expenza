## ⚡ Installation (2 minutes)

### Option 1: Quick Install
```bash
pip install ttkbootstrap matplotlib
python expensetracker.py
```

### Option 2: Using Requirements File
```bash
pip install -r requirements.txt
python expensetracker.py
```

---

## 📝 First Steps (3 minutes)

### 1. Add Your First Expense (30 seconds)

```
Category: Food
Amount: 250
Description: Lunch at cafe
```

Click **➕ Add Expense**

### 2. View Statistics (10 seconds)

Look at the right panel:
- Total Expenses: Rs 250.00
- Total Entries: 1
- Categories: 1

### 3. Try Search (20 seconds)

Type "food" in the search box → See your expense filtered instantly

### 4. View Charts (30 seconds)

Click **📊 View Charts** → See pie and bar charts of your spending

### 5. Generate Report (20 seconds)

Click **📋 Generate Report** → Select "Daily Report" → See today's total

### 6. Export Data (30 seconds)

Click **📤 Export** → Choose location → Save as CSV or JSON

---

## 🎯 Common Tasks

### Add Multiple Expenses Quickly

1. Enter first expense details
2. Click **➕ Add Expense**
3. Fields clear automatically
4. Enter next expense
5. Repeat!

### Edit a Mistake

1. Click on the wrong expense in the table
2. Click **✏️ Edit**
3. Modify the details
4. Click **➕ Add Expense** to save

### Delete an Expense

1. Click on the expense
2. Click **🗑️ Delete**
3. Confirm deletion

### Find Specific Expenses

**By Search**:
- Type any keyword in search box
- Results filter instantly

**By Category**:
- Use category dropdown
- Select specific category
- Click "Reset View" to see all again

## 💡 Pro Tips

1. **Use Tab Key**: Quickly move between fields
2. **Consistent Categories**: Use same names (Food, not food/FOOD)
3. **Daily Updates**: Add expenses every day
4. **Weekly Exports**: Backup your data regularly
5. **Check Charts**: Review spending patterns weekly

---

## 🆘 Quick Troubleshooting

| Problem | Solution |
|---------|----------|
| "Fill all fields" warning | Enter text in all 3 fields |
| "Invalid amount" error | Use only numbers (250 or 250.50) |
| Search shows nothing | Check spelling or click "Reset View" |
| Charts not showing | Add at least one expense first |
